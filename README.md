**Image Steganography Application**

**Overview**
This project is an Image Steganography Application built using Python. It allows users to hide secret messages within images and retrieve hidden messages from images. The application features a graphical user interface (GUI) created with Tkinter, making it easy to use.

**Features**

1.Encode Messages: Hide secret messages within images.
2.Decode Messages: Retrieve hidden messages from images.
3.User-friendly GUI: Built with Tkinter for a smooth user experience.
4.Supports Multiple Image Formats: Works with PNG, JPEG, and BMP images.

**Prerequisites**
To run this application, you need to have Python installed on your machine.
